# موقع بيان POS - Bayan Website

الموقع الرسمي والتعريفي لنظام كاشير وإدارة الأعمال الأقوى **Bayan POS**.

---

## 🚀 كيفية تشغيل واستضافة الموقع على GitHub Pages

لاستضافة هذا الموقع مجانًا على **GitHub Pages**، اتبع الخطوات التالية:

### الخطوة 1: رفع المشروع على GitHub
1. قم بإنشاء مستودع جديد (New Repository) على حسابك في GitHub باسم `Bayan_Website` أو أي اسم تفضله.
2. قم برفع جميع ملفات هذا المجلد إلى المستودع (بما فيها ملفات `index.html` و `style.css` و `script.js` و `version.json` وملف `.nojekyll` ومجلد `images`).

### الخطوة 2: تفعيل الاستضافة (GitHub Pages)
1. افتح صفحة المستودع الخاص بك على موقع GitHub.
2. اذهب إلى تبويب الإعدادات **Settings** (أيقونة الترس في القائمة العلوية للمستودع).
3. من القائمة الجانبية اليسرى، اختر **Pages** (تحت قسم Code and automation).
4. تحت قسم **Build and deployment**:
   - تأكد من أن خيار **Source** مضبوط على **Deploy from a branch**.
   - تحت قسم **Branch**: اختر الفرع الرئيسي للمستودع (غالباً يكون اسمه `main` أو `master`).
   - اترك المجلد المختار كما هو `/ (root)` ثم اضغط على زر حفظ **Save**.

### الخطوة 3: تصفح الموقع
- بعد الحفظ، انتظر دقيقة أو دقيقتين حتى تكتمل عملية البناء (Deployment).
- سيظهر لك رابط موقعك المباشر في نفس الصفحة بالأعلى (مثال: `https://username.github.io/repository-name/`).

---

## 🛠️ معلومات المطورين
- **ملف الإصدار (`version.json`)**: يحتوي على رقم الإصدار الحالي للموقع للتحكم في الكاش وتنزيل التحديثات فوراً للمتصفحات. عند إجراء أي تعديل مستقبلي، قم بتغيير الرقم في هذا الملف ليقوم الموقع بعمل تحديث تلقائي للمستخدمين.
- **ملف سياسة الخصوصية (`privacy.html`)**: يحتوي على سياسة الخصوصية للمستخدمين ومتوافق مع النمط البصري العام للموقع.
